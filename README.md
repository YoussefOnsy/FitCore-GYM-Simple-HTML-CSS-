# FitCore Gym

A modern and responsive fitness website built with HTML and CSS. The project showcases a premium gym experience through an attractive landing page design, highlighting gym services, trainers, testimonials, membership plans, and a registration form.

## Features

- Modern and clean user interface
- Hero section with call-to-action buttons
- Services and benefits section
- Gym statistics and achievements
- Customer testimonials section
- Membership registration form
- Footer with contact information and quick navigation links

## Sections

- Home / Hero Section
- Why Choose FitCore Gym
- Services & Facilities
- Member Testimonials
- Statistics Overview
- Membership Registration Form
- Contact Information & Footer

## Technologies Used

- HTML5
- CSS3

## Project Goals

This project was developed to:

- Practice modern HTML and CSS techniques
- Create an attractive landing page for a fitness business
- Demonstrate UI/UX design principles


## 🔮 Future Improvements

- Add JavaScript interactivity
- Implement dark/light mode
- Connect the membership form to a backend service
- Add membership pricing plans
- Integrate Google Maps for gym location
- Add animations and scroll effects
